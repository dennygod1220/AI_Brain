# 研究清單

| 名稱 | 功能簡介 | 連結 |
| :--- | :--- | :--- |
| **TimesFM** | Google 開發的時間序列基礎模型，用於進行零樣本 (Zero-shot) 的數值預測。 | [GitHub](https://github.com/google-research/timesfm) |
| **The Agency (agency-agents)** | 大量預製的 AI agent 人才庫，包含多個 division（engineering、marketing、project-management 等）。每個 agent 為 Markdown 檔，具 persona、流程與交付範例，可透過提供的 scripts 整合到 Claude Code、OpenCode、Gemini CLI 等工具；適合用作 Hermes/agent SOUL 與 skills 轉換的研究樣本。 | [GitHub](https://github.com/msitarzewski/agency-agents) |
| **nuwa-skill（女娲）** | 把名人的「思维方式」蒸馏成 Claude Code Skill 的專案。輸入名字自動调研、提煉心智模型/決策啟發式/反模式，生成可安裝的 Skill。已蒸馏 13 位人物（乔布斯、马斯克、芒格、Naval、Karpathy 等）+ 1 個主題（X 導師）。配套 darwin-skill 自動優化。12.5k stars。 | [GitHub](https://github.com/alchaincyf/nuwa-skill) |
