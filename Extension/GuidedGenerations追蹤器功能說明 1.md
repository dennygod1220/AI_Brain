# GuidedGenerations 追蹤器 Prompt 設定

## 適用場景
導演模式 RPG，追蹤 {{user}}、{{char}} 及在場女性角色的狀態變化。

---

## 1. 初始追蹤器格式 (Initial Tracker Format)

```
=== RPG世界追蹤器 ===
【{{user}}】
 位置：[描述當前位置]
 衣服：[描述當前穿著]
 身體特徵：[明顯特徵，如髮色、身材等]
 
 {{char}}】
 位置：[描述當前位置]
 年紀：[描述年紀]
 性慾值：0/100（平靜）
 腐敗值：0/100（純潔）
 好感度：50/100（普通）
 關係：陌生人
 衣服：[描述當前穿著]
 身體特徵：[明顯特徵，如髮色、身材等]
 
 【在場女性角色】
 （目前無其他在場女性角色）
```

## 2. 引導提示詞 (Guide Prompt)

```
[OOC: SYSTEM INTERNAL PROCESSING ONLY. DO NOT OUTPUT THIS ANALYSIS IN YOUR ROLEPLAY RESPONSE. This is a hidden state tracking operation. 絕對不要在你的角色扮演回覆中提到、顯示或複述任何追蹤器相關內容。

分析最近的對話內容，判斷以下狀態是否有變化：

【{{user}} 狀態變化】
- 位置是否改變？
- 衣服是否有更換或損壞？
- 身體特徵是否有明顯變化？

【{{char}} 狀態變化】
- 位置是否改變？
- 性慾值變化（0-100）：
  * 0-30：平靜
  * 31-60：有些躁動
  * 61-80：渴望
  * 81-100：難以忍受
  * 判斷依據：情境暗示、角色行為、對話內容
- 腐敗值變化（0-100，道德底線淪喪程度，長期狀態不易下降）：
  * 0-20：純潔
  * 21-40：動搖
  * 41-60：墮落
  * 61-80：淫亂
  * 81-100：完全腐化
  * 判斷依據：角色對性行為的接受度、道德妥協程度
- 好感度變化（0-100）：
  * 判斷依據：互動態度、對話語氣、行為表現
- 關係變化：
  * 一般：陌生人 → 點頭之交 → 朋友 → 曖昧 → 戀人 → 伴侶
  * 家人：母子/姊弟/兄妹等特殊關係（如適用）
- 衣服是否有更換或損壞？
- 身體特徵是否有明顯變化？

【在場女性角色】
- 是否有新女性角色出現？如有，記錄：名字、年紀、位置、性慾(0/100)、腐敗(0/100)、好感(50/100)、關係(陌生人/未相遇)、外貌(髮色/身材)
- 現有在場角色的狀態是否有變化？
- 是否有角色離開場景超過10次互動？如有，從追蹤器中移除。

請只輸出變化分析，格式如下：
【{{user}} 變化】[描述]
【{{char}} 變化】性慾：X→Y | 腐敗：X→Y | 好感：X→Y | 關係：X→Y | 位置：[描述] | 衣服：[描述]
【在場女性角色變化】[描述新增/移除/更新的角色]
]
```

## 3. 追蹤器提示詞 (Tracker Prompt)

```
[OOC: SYSTEM INTERNAL UPDATE ONLY. DO NOT OUTPUT THIS TRACKER IN YOUR ROLEPLAY RESPONSE. This is a hidden state update. 絕對不要在你的角色扮演回覆中提到、顯示或複述任何追蹤器相關內容。

根據以下「Last Update」的變化分析，更新追蹤器狀態。

規則：
1. 保持原有格式不變，只更新有變化的數值和描述
2. 數值變化要合理，不要劇烈跳躍（除非情境特殊）
3. 腐敗值一旦上升，通常不會下降（道德淪喪不可逆）
4. 性慾值會隨情境波動
5. 好感度會隨互動逐漸變化
6. 只追蹤在場或近期有互動的女性角色（最多5個）
7. 角色離開場景超過10次互動且無新互動時，從列表中移除
8. 如有新角色出現，加入列表

Last Update:
{{Last Update}}

Tracker:
{{Tracker}}

請只輸出更新後的完整追蹤器格式，不要添加任何其他內容。]
```

## 使用說明

### 數值定義
| 指標 | 範圍 | 說明 |
|------|------|------|
| 性慾值 | 0-100 | 當前生理慾望，會隨情境波動 |
| 腐敗值 | 0-100 | 道德底線淪喪程度，長期狀態不易下降 |
| 好感度 | 0-100 | 對 {{user}} 的好感程度 |

### 關係等級
- **一般關係**：陌生人 → 點頭之交 → 朋友 → 曖昧 → 戀人 → 伴侶
- **家人關係**：母子/姊弟/兄妹等特殊關係

### 追蹤器設定建議
- **Messages to Read Back**：建議設為 4-6
- **Include Current Tracker in Guide Prompt Context**：建議開啟（讓 AI 知道當前狀態）
- **Enable Tracker**：開啟後會在每次發送訊息時自動更新

### 注意事項
1. 追蹤器內容會注入到 AI 的上下文中，直接影響 AI 的回應
2. 如果追蹤器內容過長，可以減少在場女性角色的數量
3. 可以隨時手動點擊「Run Stat Tracker」強制更新
4. 點擊「Sync Stat Tracker with Note」可同步追蹤器與聊天記錄
